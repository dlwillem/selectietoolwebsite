Installatie

Stap voor stap aan de slag met de SelectieTool op je eigen server.

  • 1
    Clone de repository git clone git@github.com:dlwillem/st.git
  • 2
    Installeer dependencies via Composer composer install --no-dev --optimize-autoloader
  • 3
    Open de installatie-wizard in de browser https://jouwdomein.nl/install.php

    De wizard begeleidt je in 7 stappen door database-setup, admin-aanmaak en optionele seed-data.

  • 4
    Verwijder install.php na installatie

    Laat dit bestand niet staan op productie.

  • 5
    Log in met het admin-account

    Gebruik het account dat je in stap 3 hebt aangemaakt via de wizard.
